# Project Title : 편의시설 안내 서비스


# Project Description : 

+학생들이 학교 내 편의시설과 캠퍼스 시설을 더욱 쉽게 이용할 수 있도록 돕는 서비스입니다. 화장실, 정수기, 샤워실 등 다양한 편의시설의 **위치**를 안내합니다. 사용자는 **지도를 터치**하여 캠퍼스 내 시설의 **이용시간 및 상세 정보**를 확인할 수 있습니다.
